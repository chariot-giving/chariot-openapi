<Tip title="Sandbox Key Prerequisite">
Before using this implementation guide, ensure that your organization has been provided with a API Key for our sandbox environment. If you have not yet received your sandbox keys, please reach out to support@givechariot.com for assistance.
</Tip>

<Warning title="Production Key Requirements">
All items on this implementation checklist must be completed before your organization can receive production keys. Once everything is complete, please fill out the production keys request form to proceed.
</Warning>
