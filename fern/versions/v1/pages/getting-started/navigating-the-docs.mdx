The Chariot documentation is broken up into three main parts: 

1. [Quick Start](quickstart): Integrating with Chariot Made Easy. This is a guide meant to provide an overview of the integration process and get you started quickly.
1. [Integrating DAFpay](/guides/dafpay/overview): The development guide for the frontend, client-side component that launches [DAFPay](https://www.dafpay.com/): the application that users interact with in order to authenticate with and initiate grant requests from their Donor Advised Funds. Both React and Javascript SDKs are available. 
2. [Using the API](/api/overview): The development guide for the backend APIs needed to register new nonprofits, submit grants, and query for transactional data and events. All endpoints much be authenticated using OAuth 2.0 Client Credentials Flow. SDKs are available to streamline development efforts.
3. [Implementation Guide](/guides/dafpay/implementation): A comprehensive guide for product success; a collection of best practices and strategies for successfully integrating DAFpay into your platform and user experience.
