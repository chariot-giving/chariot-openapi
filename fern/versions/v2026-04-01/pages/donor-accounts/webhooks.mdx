Subscribe to these events via an [Event Subscription](/api/event-subscriptions) to drive your internal workflows. All events include an `associated_object_id` you can use to fetch the full object.

## Account Events

| Event | When it fires |
|---|---|
| `donor_account.created` | A new Account has been created. In the Donor-Initiated Verification flow, this is your signal that a donor has submitted a grant and is waiting for verification. |
| `donor_account.approved` | The Account transitioned to `approved` — either because a token was verified or because the donor consumed a DAF-Initiated token in DAFpay. |
| `donor_account.rejected` | The Account transitioned to `rejected`. All open Grant Requests tied to it are automatically rejected. |
| `donor_account.updated` | Any other update to the Account (e.g. `external_id` or `metadata` changed). |

## Grant Request Events

| Event | When it fires |
|---|---|
| `grant.created` | A new Grant Request was submitted via DAFpay. May arrive before `donor_account.created` if both are created in the same flow. |
| `grant.approved` | A Grant Request was approved for processing. |
| `grant.rejected` | A Grant Request was rejected. |

## Setting up an Event Subscription

<CodeBlocks>
```bash cURL
curl -X POST https://api.givechariot.com/v1/event_subscriptions \
  -H "Authorization: Bearer $CHARIOT_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "url": "https://your-server.example.com/webhook",
    "enabled_events": [
      "donor_account.created",
      "donor_account.approved",
      "donor_account.rejected",
      "donor_account.updated",
      "grant.created",
      "grant.approved",
      "grant.rejected"
    ]
  }'
```
```typescript TypeScript
await chariot.eventSubscriptions.create({
  url: "https://your-server.example.com/webhook",
  enabled_events: [
    "donor_account.created",
    "donor_account.approved",
    "donor_account.rejected",
    "donor_account.updated",
    "grant.created",
    "grant.approved",
    "grant.rejected",
  ],
});
```
</CodeBlocks>
