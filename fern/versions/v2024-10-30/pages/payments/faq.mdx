## What if a donor wants to grant to a nonprofit that isn't on Chariot's Organization Search?
Chariot supports the ability to add a new organization to the system. Please call the [INSERT API ENDPOINT] to submit a request for
an organization to be reviewed for addition.

## What are nonprofits charged for using Chariot Payments?
Nonprofits are not charged for using Chariot Payments. Chariot charges a transaction fee to the grantmaker.
For nonprofits that want additional services in supporting their gift processing needs, Chariot has subscriptions plans available.
